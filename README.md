# Sharon Odamtten-Sowah Portfolio

A premium static portfolio website for Sharon Odamtten-Sowah.

## GitHub Pages Setup

1. Upload `index.html`, `style.css`, `script.js`, and the `assets` folder to the root of the repository.
2. Go to Settings > Pages.
3. Select Deploy from a branch.
4. Choose `main` and `/root`.
5. Save and wait for the site link.
